\chapter{Introduction}

Software is written with intent, an intent to make a computer do something. Software
engineering, in essence, is the process of taking such intent, refining, reshaping,
reformulating it into a form that the computer understands. Software testing is the
bridge between the intent and the produced program; it is the process of tinkering,
measuring, understanding what the program does, separating it from how the program is
implemented.
